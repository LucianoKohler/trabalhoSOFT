# BRUTE Bowling
Este repositório representa o trabalho final da matéria de Engenharia de Software do nosso curso de Ciência da Computação na UDESC. Nele, realizamos todas as etapas de modelagem de um sistema de gestão de pistas de boliche e máquinas de fliperama, documentando:

1. Descrição do Problema
2. Levantamento de regras de negócio, requisitos funcionais e não funcionais
3. Estimativas de tempo de desenvolvimento do projeto
4. Diagramas de classe (UML)

Todas as informações estão presentes no documento [.PDF contido no repoistório]()!
